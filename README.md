# De Pátria para Pátria

## Descrição do projeto

Este projeto é uma página web responsiva desenvolvida como parte do Sprint 5 do curso. A proposta foi criar uma galeria digital chamada "De Pátria para Pátria", apresentando histórias e origens de profissionais ao redor do mundo que fazem parte da comunidade TripleTen.

A página contém:

1. Um header com o logotipo;
2. Uma seção principal com título, subtítulo e imagem;
3. Uma seção introdutória com texto explicativo e citação;
4. Estrutura preparada para a seção de cidades; e
5. Rodapé com informações autorais.

O principal objetivo do projeto foi praticar a construção de layouts fiéis ao Figma e totalmente responsivos para diferentes tamanhos de tela (desktop, tablet e mobile).

## Tecnologias

1. HTML5;
2. CSS3;
3. Metodologia BEM;
4. Git e GitHub; e
5. GitHub Pages.

## Técnicas utilizadas

1. Estruturação semântica com header, main, section e footer;
2. Organização de arquivos em blocos (arquitetura em pastas);
3. Responsividade:

- 1024 (tablet);
- 544px (mobile);

4. Uso de max-width para manter o layout fluido;
5. Uso de line-height proporcional (valores decimais); e
6. Layout fluido sem scroll horizontal.

O foco principal foi manter o design o mais fiel possível ao layout do Figma e garantir que o site funcione corretamente em larguras intermeditárias, como 470px, 768px, 999px e 1100px.
